# 智能个人作品集系统 - 测试计划

## 1. 测试目标

确保系统在公开展示、后台管理、JD 智能适配、模块实验室、分享链接和部署流程中具备基本可用性、稳定性和安全性。

## 2. 测试范围

- 前端页面展示
- 前端交互
- 后端接口
- 数据库读写
- 认证鉴权
- JD 智能适配
- 工作模块复现 Demo
- 访问统计
- Docker 部署
- Jenkins 流水线

## 3. 前端测试

### 3.1 页面验收

需要覆盖：

- 首页
- 在线简历页
- 项目列表页
- 项目详情页
- 模块实验室首页
- 视频学习 Demo
- JD 智能适配页
- 面试模式页
- 后台登录页
- 后台管理页

验收点：

- 桌面端布局正常。
- 移动端无明显溢出。
- 文案不被截断。
- 页面加载、空状态、错误状态完整。

### 3.2 交互测试

- 导航跳转。
- 项目筛选。
- Demo 操作。
- JD 输入和提交。
- Variant 预览。
- 后台表单新增、编辑、删除。

## 4. 后端测试

### 4.1 单元测试

重点：

- JD Prompt 构造。
- Variant 生成。
- Demo 状态计算。
- 分享链接有效性判断。
- 统一响应和异常处理。

### 4.2 接口测试

重点接口：

- 登录接口。
- 首页数据接口。
- 项目详情接口。
- ModuleDemo 接口。
- JD 分析接口。
- 分享链接接口。
- 访问统计接口。

### 4.3 鉴权测试

- 未登录不能访问后台接口。
- Token 过期后不能继续访问。
- 公开接口不需要登录。
- 管理接口必须鉴权。

## 5. JD 智能适配测试

测试用例：

- 输入 Java 后端 JD。
- 输入 AI 应用 JD。
- 输入前端 JD。
- 输入过长 JD。
- 输入空 JD。
- LLM 调用失败。
- LLM 返回非 JSON。

验收点：

- 能返回结构化结果。
- 有 `supportedBy` 和 `riskNotes`。
- 不编造不存在的经历。
- 失败时有明确错误提示。

## 6. 工作模块复现测试

### 6.1 视频学习 Demo

测试用例：

- 正常播放并上报进度。
- 暂停后不继续上报。
- 拖拽进度。
- 重复上报。
- 完播同步。
- 重置演示数据。
- 模拟接口失败。

验收点：

- Redis 快照、MySQL 快照、状态变化展示合理。
- 日志顺序清晰。
- 前端 UI 状态与后端返回一致。

### 6.2 Agent 工作流 Demo

测试用例：

- SQL 问答流程。
- RAG 问答流程。
- 节点失败。
- 节点重试。

验收点：

- 每个节点有输入、输出、状态、耗时。
- 失败场景有兜底提示。

## 7. 部署测试

### 7.1 Docker Compose

- 所有容器能启动。
- web 能访问。
- server 能访问。
- MySQL 连接正常。
- Redis 连接正常。

### 7.2 Nginx

- `/` 转发前端。
- `/api` 转发后端。
- 健康检查通过。

### 7.3 Jenkins

- 能拉取代码。
- 前端构建成功。
- 后端测试和打包成功。
- Docker 镜像构建成功。
- 部署后健康检查通过。

## 8. 回归测试清单

每次重要改动后检查：

- 首页是否正常。
- 项目详情是否正常。
- Demo 是否可运行。
- JD 分析是否可提交。
- 后台是否可登录。
- Docker 是否可启动。

## 9. 验收标准

第一阶段验收：

- 本地启动成功。
- 主要页面可访问。
- 至少一个 Demo 可交互。
- JD 适配基础流程可用。
- 后端接口有 Swagger。
- Docker Compose 可启动。

上线验收：

- 域名访问正常。
- HTTPS 正常。
- Jenkins 自动部署成功。
- 后台管理可用。
- 关键页面移动端可用。
